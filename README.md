# Proxy Telegram Bot API Service

```sh
# method = post
# uri = 'http://localhost:2456/sendMessage'
# body.token = 'xxxx'
# body.text = 'Hello World'
# body.chat_id = '123465xxx'
```


- npm

  ```bash
  npm install
  npm run serve
  ```

- yarn

  ```bash
  yarn
  yarn serve
  ```

- pnpm
  ```bash
  pnpm install
  pnpm run serve
  ```
